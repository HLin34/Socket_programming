# Introduction to Computer Networks Final Project 2023

## Environment Requirements: Ubuntu 22.04

## Getting Started

1. use tmux and use CTRl+B+SHIFT+5 to set two sessions of terminal and show on the screen at the same time.

2. Compile the file manually. 
   
   One session compile the server.cpp:
   => g++ server.cpp -o server
   => ./server

   One session compile the client.cpp 
   => g++ client.cpp -o client
   => ./client
